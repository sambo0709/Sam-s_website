Event Registration

Technologies Used

  • Node.js
  • Express
  • MongoDB
  • Mongoose
  • REST APIs

Features

  • RESTful API endpoints for event data
  • Database integration with schema modeling
  • User registration workflow handling
  • Structured backend routing

What I Learned

This project strengthened my understanding of RESTful API design, backend routing, and database schema modeling with MongoDB and Mongoose. I also gained experience structuring scalable server-side applications and handling real-world data flows between client and database layers.
